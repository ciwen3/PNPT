Функция FreeRouter

Закрывает открытый дескриптор объекта Router.

Примечание  После вызова этой функции, указанный дескриптор более не является действительным.

Синтаксис

Delphi
C++
function FreeRouter(
  hRouter: THandle
): Bool; stdcall;

BOOL WINAPI FreeRouter(
  _In_  HANDLE hRouter
);

Параметры

hRouter [in]

Действительный дескриптор объекта Router. Он может быть получен при помощи функции PrepareRouter.

Возвращаемое значение

Если функция выполнилась успешно, возвращается ненулевое значение.

Если же функция выполнилась неудачно, возвратится нуль.
