Функция IsRouterStopping

Проверяет, получил ли обработчик роутеров сообщение об отмене обработки указанного устройства.

Примечание  Чтобы отправить обработчику роутеров сообщение об отмене, используйте функцию StopRouter.

Синтаксис

Delphi
C++
function IsRouterStopping(
  hRouter: THandle
): Bool; stdcall;

BOOL WINAPI IsRouterStopping(
  _In_  HANDLE hRouter
);

Параметры

hRouter [in]

Действительный дескриптор объекта Router. Он может быть получен при помощи функции PrepareRouter.

Возвращаемое значение

Если обработчик роутеров получил сообщение об отмене сканирования указанного устройства, функция вернёт true.

В противном случае функция вернёт false.
