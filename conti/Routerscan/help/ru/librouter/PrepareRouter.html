Функция PrepareRouter

Выполняет подготовку к сканированию выбранного устройства.

Синтаксис

Delphi
C++
function PrepareRouter(
  Row,
  IP: DWORD;
  Port: Word;
  var hRouter: THandle
): Bool; stdcall;

BOOL WINAPI PrepareRouter(
  _In_   DWORD Row,
  _In_   DWORD IP,
  _In_   WORD Port,
  _Out_  HANDLE *hRouter
);

Параметры

Row [in]

Номер строки таблицы, который будет использоваться для получения информации о выбранном устройстве.

IP [in]

32-битный IPv4 адрес устройства.

Port [in]

TCP порт устройства.

hRouter [out]

Дескриптор объекта Router, который может быть использован в родственных функциях.

Возвращаемое значение

Если функция выполнилась успешно, возвращается ненулевое значение.

Если же функция выполнилась неудачно, возвратится нуль.
