Функция ScanRouter

Запускает обработку указанного устройства.

Примечание  Эта функция вернёт управление только после окончания обработки устройства.

Синтаксис

Delphi
C++
function ScanRouter(
  hRouter: THandle
): Bool; stdcall;

BOOL WINAPI ScanRouter(
  _In_  HANDLE hRouter
);

Параметры

hRouter [in]

Действительный дескриптор объекта Router. Он может быть получен при помощи функции PrepareRouter.

Возвращаемое значение

Если функция выполнилась успешно, возвращается ненулевое значение.

Если же функция выполнилась неудачно, возвратится нуль.
