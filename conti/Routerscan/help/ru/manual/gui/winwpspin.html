Помощник WPS PIN.


Рис 1. Окно помощника WPS PIN.

Помощник может подсказать, какой пин-код использовать для необходимого устройства. В нём реализовано множество алгоритмов генерации WPS пин-кода на основе MAC адреса устройства, а также множество ассоциаций различных MAC адресов с типами устройств и алгоритмами генерации.

    Содержание

  • 1. Список BSSID для проверки
  • 2. Управляющие кнопки
  • 3. Область вывода информации
  • 4. Реализованные алгоритмы генерации
  • 5. Проверка алгоритмов в таблице

• Список BSSID для проверки

Сюда вводится один или несколько BSSID интересующих точек доступа.

• Управляющие кнопки

- [Suggest] - предположить пин-код, согласно подходящему алгоритму по первой части MAC адреса. Если к MAC адресу ни один алгоритм не подошёл, пин-код для него сгенерирован не будет.
- [Get All] - сгенерировать все возможные пин-коды, не зависимо от того, подходят они или нет. Работает только с одним MAC адресом.
- [Grab from JSW] - получает список BSSID из таблицы запущенной программы JumpStart for Wireless от Atheros (также известной как TP-LINK QSS for Wireless).

• Область вывода информации

Сюда выводится информация о пин-коде для MAC адреса, либо техническая информация об алгоритме.

• Реализованные алгоритмы генерации

Позволяет посмотреть, какие алгоритмы поддерживаются генератором, техническая информация и внешние ссылки на них.

Примечание: Список поддерживаемых алгоритмов может изменяться в разных версиях программы.

• Проверка алгоритмов в таблице

Если в выбранной таблице есть устройства с пин-кодами, данная функция позволит проверить, какие пин-коды соответствуют реализованным алгоритмам генерации. Результаты будут выведены в колонку комментариев.
